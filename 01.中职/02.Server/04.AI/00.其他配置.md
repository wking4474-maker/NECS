1. 在Linux8中为ollama配置用户组，该用户仅用于运行服务，禁止其登录系统 ，家目录位于/usr/share/ollama。
```shell

groupadd ollama

useradd -g ollama -d /usr/share/ollama -s /sbin/nologin ollama

```

2. 在 Linux8 中为root用户加入附加组ollama。
```shell
usermod -aG ollama root
```

3. 在 Linux8 中将文件ollama-linux-arm64.tar.zst解压安装至/usr目录下。

4. 在 Linux8 中编写 SystemD 服务配置，使用服务运行专用用户完成启动，无论发生任何意外情况，服务都会进行自动重启（冷却3秒)，以保证服务可靠运行，服务需要监听所有地址。
```shell
vim /etc/systemd/system/ollama.service

[Unit]
Description=Ollama Service
After=network-online.target
Wants=network-online.target

[Service]
Type=simple

User=ollama
Group=ollama

# 监听所有地址（必要）
Environment="OLLAMA_HOST=0.0.0.0:11434"

# 数据/模型目录（推荐：统一放这里）
Environment="OLLAMA_HOME=/root/.ollama"
Environment="OLLAMA_MODELS=/root/.ollama/models"

WorkingDirectory=/root/.ollama
ExecStart=/opt/bin/ollama serve

# 任何情况都重启 + 冷却3秒
Restart=always
RestartSec=3s

[Install]
WantedBy=multi-user.target

systemctl daemon-reload
systemctl start ollama
```
4. 在 Linux8 中启动ollama服务，并完成开机启动配置，最终启动ollama服务。
```shell
systemctl enable ollama
```
