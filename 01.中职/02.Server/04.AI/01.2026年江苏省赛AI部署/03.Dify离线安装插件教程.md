## 1.本地下载插件

## 2.dify插件打包工具
```shell
git clone https://github.com/junjiem/dify-plugin-repackaging.git
cd dify-plugin-repackaging
```

## 3.打包插件
- 将下载的插件放到dify-plugin-repackaging/目录里,执行打包脚本
```shell
./plugin_repackaging.sh local ./待打包插件名.difypkg
```
- 生成离线插件`待打包插件名_offline.difypkg`

## 4.离线安装

- 在服务器上dify项目修改 .env 配置文件，在文件末尾添加以下参数
```shell
FORCE_VERIFYING_SIGNATURE=false		
# Dify 平台允许安装所有未在 Dify Marketplace 上架（审核）的插件

PLUGIN_MAX_PACKAGE_SIZE=524288000 	
# Dify 平台允许安装 500M 大小以内的插件

NGINX_CLIENT_MAX_BODY_SIZE=500M 	
# Nginx客户端将允许上传 500M 大小以内的内容

```

- 重启dify

```shell
docker compose down
docker compose up -d
```

- dify插件安装选择本地插件，选择文件，即安装成功
